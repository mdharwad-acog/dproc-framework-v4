DProc

End-User Guide

Run AI-powered data processing pipelines

What is DProc?

  • A tool that runs data processing pipelines
  • Uses AI (Claude, GPT-4, Gemini) to analyze data
  • Generates reports in multiple formats
  • Works via command line or web interface

Two Ways to Use DProc

Web Interface

Easy to use. Just fill forms and click "Run"

Command Line

For technical users who prefer terminal

Running a Pipeline (Web)

  • Open the web interface at localhost:3000
  • Select a pipeline from the list
  • Fill in the input fields (varies by pipeline)
  • Choose output format (HTML, PDF, Markdown)
  • Click "Run" and wait for results

Running a Pipeline (Command Line)

  • Use: dproc list to see available pipelines
  • Use: dproc execute pipeline-name --input '{...}'
  • Check results with: dproc history
  • View stats with: dproc stats pipeline-name

Checking Your Results

  • Reports are generated in your chosen format
  • View execution history to track all runs
  • Each execution has a unique ID for reference
  • Output files are saved automatically
